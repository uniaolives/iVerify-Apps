$$\hat{H}_{\text{total}}|\Psi\rangle = 0$$

Where the total Hamiltonian is decomposed into:
$$\hat{H}_{\text{total}} = \hat{H}_{\text{clock}} \otimes \hat{I}_{\text{system}} + \hat{I}_{\text{clock}} \otimes \hat{H}_{\text{system}}$$

Time evolution emerges when we condition the system state on a specific clock state $|\tau\rangle$:
$$|\psi(\tau)\rangle_{\text{system}} = \langle\tau|_{\text{clock}} \otimes \hat{I}_{\text{system}} |\Psi\rangle$$
